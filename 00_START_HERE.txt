                         📞 QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

❌ Port 8000 already in use?
   → Close other Django instances
   → Or change port in run_all_servers.py

❌ Port 5055 already in use?
   → Close other RASA instances
   → Or change port in run_all_servers.py

❌ RASA won't start?
   → Run: pip install rasa rasa-sdk
   → Check Python version: python --version (need 3.8+)

❌ Django won't start?
   → Run: pip install -r requirements.txt
   → Check DEBUG=True in .env file

❌ Can't find START_SERVERS.bat?
   → Navigate to: C:\Users\LENOVO\OneDrive\Desktop\GovScheme
   → It's in that folder
